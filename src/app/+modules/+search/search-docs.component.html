Directives / Components

A set of directives and components that work together to create a search component.

Usage Description
.ui.search <div class="ui search" [source]="basicSource" ></div> The container for the component of the search. *Note the results div is not needed because it is added by this component.
input.prompt <input type="search" class="prompt" /> The search input that will be used for this search component.

Settings

Settings that can be added to the elements to change the functionality of the search component.

Default Description
[transition] fade The animation transition that should be used when this element is displayed and hidden.
[duration] 200 The duration of the transition.
[source] The local search object to be used for searching.
[maxResults] 7 The number of max results that should be displayed when the search is done.
[searchCallback] The search callback that should be called instead of the internal callback.
[debounce] 400 The number of milliseconds that should be delayed before searching will take place.
