Directives / Components

A set of directives that work together to create progress bars.

Usage Description
.ui.progress <div class="ui progress"></div> The progress declaration.
.bar <div class="bar"></div> The bar element that will be updated with the progress meter.
ngx-bar-progress-label.progress <ngx-bar-progress-label class="progress"></ngx-bar-progress-label> Internal label for the bar component. This will be injected into the bar element.

Settings

Settings that can be added to the elements to change the functionality of the progress bars.

Default Context Description
[progress] *required .ui.progress This is the progress in which the bar should be rendered.
[label] visible .ui.progress Determines if the label should be visible on the progress bar.
[autoSuccess] true .ui.progress Determines if the success class should be added once 100% has been reached.
[total] 100 .ui.progress The overall total used to calculate the progress.
