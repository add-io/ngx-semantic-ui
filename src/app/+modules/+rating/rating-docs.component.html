Rating

A rating indicates user interest in content.
Examples Settings

Basic Rating

Ratings are counted as an input and can be used as such.

Rating: {{basicRating}}

Icon Variations

Ratings can take on different icons to use for the rating system.

Rating: {{starRating}}
Rating: {{heartRating}}

Size Variations

Ratings can also come in different sizes.

Components

A set of components that will build out the rating system.

Usage Description
.ui.rating <div class="ui rating" data-rating="10"></div> The rating wrapper that will build out the rating icons and create the input for that element.

Settings

Settings that can be added to the elements to change the functionality of the rating component.

Default Description
[data-rating] | [rating] 4 The amount of rating icons the rating object will have.
[clearable] true Clearable if the current rating is selected twice.
