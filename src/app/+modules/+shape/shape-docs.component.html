Directives / Components

A set of directives and components that work together to create shapes.

Usage Description
.ui.shape <div class="ui shape"></div> This is the component that will create a shape that can be flipped by calling various methods.
.sides <div class="sides"></div> The container for all the sides of the shape being rendered.
.side <div class="side"></div> One of the sides of the shape that should be rendered, one should be active before shape will work.

Settings

Settings that can be added to the elements to change the functionality of the sticky div.

Default Context Description
[duration] 500 .ui.shape The duration of the animations when transiting to other sides.
