Directives / Components

A set of directives and components that work together to create modals

Usage Description
.ui.modal <div class="ui modal" #modal></div> This is the modal that will be displayed once the open method is called.
[modalContext] <div modalContext></div> The context in which the modal should be embedded into, the default is the body tag.
.positive,.approve,.ok <a class="ui button positive green"></a> Button that will trigger an approve event and close the modal.
.negative,.deny,.cancel <a class="ui button negative red"></a> Button that will trigger a deny event and close the modal.
.close.icon <i class="close icon"></i> Button that will close the modal.

Settings

Settings that can be added to the elements to change the functionality of the modal.

Default Description
[observeChanges] true Will observe the DOM of the modal and update the position to fit the screen.
[allowMultiple] false If set to true will allow multiple modals to be visible at once.
[closable] true Will close the modal if the dimmer is clicked.
[transition] Scale The transition the modal should use when opening/closing.
[duration] 400 The duration of the transition in milliseconds.

Callbacks

Callbacks that can be added to the elements to add extra functionality on the events of the modal.

Description
(show) Triggers when the modal is opened and before the animation ends.
(visible) Triggers when the modal animation to show ends.
(hide) Triggers when the modal is starting to hide but before the animation ends.
(hidden) Triggers when the modal is hidden and the animation ends.
(approve) Triggered when the approved directive is clicked.
(deny) Triggered when the deny directive is clicked.
