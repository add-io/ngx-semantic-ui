Directives

A set of directives that work together to create sticky divs.

Usage Description
.ui.sticky <div class="ui sticky"></div> This is the component that will stick to the top of the screen on scrolling.
[stickyContext] <div stickyContext></div> This is the context in which the sticky can scroll in, this mainly pertains to how far it can scroll down.

Settings

Settings that can be added to the elements to change the functionality of the sticky div.

Default Context Description
[offset] 0 .ui.sticky The offset in pixels the sticky will move down from the top of the sticky element.
