Directives

A set of directives that work together to create tabs

Usage Description
.ui.tab <div class="ui tab" data-tab="tab-path"></div> This is the content that should be displayed when the tab is active
[data-tab]:not(.ui.tab) <a class="item" data-tab="tab-path"></a> This is the trigger that will open a specific tab when clicked
[tabContext] <div tabContext="context-name"></div> This is the context this tab can trigger in, it is best to have both the tabs and the triggers

Settings

Settings that can be added to the elements to change the functionality of the tabs.

Default Context Description
[tab] | [data-tab] *required any The attribute that determines the tab content as well as the tab triggers
.active false .ui.tab | [data-tab]:not(.ui.tab) Determine if this is the default tab that should be opened if one is not selected
[history] false [tabContext] Whether the tab should be pushed to the url history of the application
