Components / Directives

A set of components and directives that work together to create dimmers.

Usage Description
[uiDimmer] <div class="ui segment" uiDimmer></div> The component that will control the dimmer for the segment.
.ui.dimmer <div class="ui dimmer"></div> The dimmer that will be overlaying the content.

Settings

Settings that can be added to the elements to change the functionality of the dimmer.

Default Description
[uiDimmer] empty The initilization to build a dimmer, however it can be bound with a boolean to control if the dimmer is shown or not.
[transition] fade The animation that should be used when opening or closing the dimmer.
[duration] 500 The duration it takes for the dimmer to finish its animation.
[closable] auto If the dimmer itself can be closed when the dimmer is clicked.
